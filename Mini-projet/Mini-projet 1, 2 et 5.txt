Mini-projet 5 : 

1. FONCTION calculerMoyenne(notes)
		POUR chaque note dans notes
			somme <- somme + note
			moyenne <- somme / taille de notes
		FIN POUR
	RETOURNER Moyenne
	FIN FONCTION

3. FONCTION meilleurNote(notes)
		noteMax <- index de notes[0]
		POUR chaque note dans notes
			SI note > noteMax
				noteMax <- note
			FIN SI
		RETOURNER noteMax
	FIN FONCTION

4. FONCTION pireNote(notes)
		noteMin <- index de notes[0]
		POUR chaque note dans notes
			SI note < noteMin
				noteMin <- note
			FIN SI
		RETOURNER noteMin
	FIN FONCTION
